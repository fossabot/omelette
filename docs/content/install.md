## :beer: Homebrew

```sh
brew install tamada/brew/omelette
```

## Go lang

```sh
go get github.com/tamada/omelette
```

## :hammer_and_wrench: Build from source codes

```sh
$ git clone https://github.com/tamada/omelette.git
$ cd omelet
$ make
```
